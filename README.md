# soundtyping
This is a program to play sound when you are typing.

### Requirements
* Ubuntu 20.04
* Python 3

### Usage
1. Download your sound file

2. Install
```bash
pip3 install git+https://github.com/TakutoYoshikai/soundtyping
```
2. Run main.py
```bash
soundtyping /path/to/sound-file
```
